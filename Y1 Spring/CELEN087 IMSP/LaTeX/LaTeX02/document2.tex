\section{Mathematics - Cases}

The absolute value function $f(x)=|x|$ is defined as:

\begin{equation*}
|x|=
\begin{cases}
x & \textrm{if} \quad x \geq 0 \\
-x & \textrm{if} \quad x \leq 0
\end{cases}
\end{equation*}
